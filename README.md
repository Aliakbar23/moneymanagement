# Forex Money Management App

A professional, single-page Forex Money Management application built with HTML, CSS, and JavaScript.

## Features

- **Trade Logger**: Record your trades with entry, SL, TP, lot size, and notes.
- **Lot Size Calculator**: Calculate the perfect position size based on your risk percentage and stop loss.
- **Statistics Dashboard**: View your win rate, total profit/loss, average R:R, and equity curve.
- **Journal**: Review your trade history with filtering options.
- **Data Management**: Export and Import your trade data (JSON) for backup.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Charting**: Chart.js (via CDN)
- **Storage**: LocalStorage (Persisted in browser)

## How to Use

1.  Clone this repository.
2.  Open `index.html` in your web browser.
3.  Start logging your trades!

## Deployment

To deploy this app to Vercel:

1.  Install Vercel CLI: `npm i -g vercel`
2.  Run `vercel` in the project directory.
